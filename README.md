PHP Scoreboard
==============




IDEA
----

Show game scoreboard with orderable score column.
| Rank          | Name           | Score  |
| ------------- |:--------------:| -----:|
| 1             | Player1 | 3300 |
| 2             | Player2        |   1000 |
| 3             | NonamePlayer   |    5 |




IMPLEMENTATION
--------------
The page is using Bootstrap 4 as a base, also using custom Javascript and JQuery. On the server-side using PHP and MySQL.
